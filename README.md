# Шаблонный класс matrix на языке C++

Реализация шаблонного класса `matrix` на языке C++, предназначенного для работы с матрицами произвольных размеров и типов данных. Класс включает различные конструкторы: по умолчанию, для создания матрицы с заданным значением, для ввода с консоли, для чтения из файла, а также конструктор копирования. Поддерживаются базовые арифметические операции (сложение, вычитание, умножение на скаляр и на другую матрицу) с проверкой совместимости размеров, перегружены операторы сравнения (включая сравнение с нулевой и единичной матрицей), а также оператор логического отрицания для вычисления обратной матрицы. Реализованы методы для нахождения определителя, транспонирования, построения матрицы алгебраических дополнений. Предусмотрена обширная функциональность для работы с файлами: чтение и запись как всей матрицы целиком, так и отдельных строк, столбцов или элементов. В коде присутствует обработка исключительных ситуаций (несоответствие размеров, вырожденные матрицы, ошибки открытия файлов). В файле `main.cpp` приведены примеры использования всех основных возможностей класса.
